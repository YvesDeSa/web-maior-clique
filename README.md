# 🕸️ Web Maior Clique

Interface web desenvolvida para interagir com a solução do problema matemático do **Maior Clique** em Grafos. Este projeto atua como o front-end (cliente) da aplicação desenvolvida originalmente para a disciplina de Programação Orientada a Objetos (POO) Aplicada.

🚀 **Acesse a aplicação online:** [web-maior-clique.vercel.app](https://www.google.com/search?q=https://web-maior-clique.vercel.app) 

## 💻 Tecnologias Utilizadas

Este projeto é um front-end estático direto ao ponto, construído com as tecnologias web clássicas:

**JavaScript (Vanilla)** (`main.js`) 
**CSS3** (`style.css`) 
**HTML5** (`index.html`) 

## ⚙️ Como executar o projeto localmente

Como a aplicação é baseada exclusivamente em arquivos estáticos (HTML, CSS e JS), não é necessário nenhum processo complexo de build ou gerenciadores de pacotes.

1. Clone este repositório para a sua máquina:
```bash
git clone https://github.com/YvesDeSa/web-maior-clique.git

```

2. Navegue até a pasta do projeto:
```bash
cd web-maior-clique

```

3. Abra o arquivo `index.html`  diretamente no seu navegador de preferência.

* *Dica:* Para uma melhor experiência de desenvolvimento e para evitar problemas com requisições assíncronas ao testar integrações, é recomendado utilizar uma extensão como o "Live Server" do VS Code.

## 🔗 Integração com a API

Este front-end foi projetado para consumir a API back-end construída em Java. Para que as funcionalidades de cálculo e visualização do grafo operem perfeitamente, certifique-se de que o back-end [grafo-maior-clique](https://www.google.com/search?q=https://github.com/YvesDeSa/grafo-maior-clique) esteja em execução e com as configurações de CORS liberadas.
